# Project Memory

This folder is the shared mailbox for Codex chats working on "Очень Большая Бродилка".

## Routine

1. Before work, read:
   - `handoff.md`
   - `updates.md`
   - your inbox in `inbox/`
2. During work, keep edits inside your assigned project folder.
3. After meaningful work, append a short update to `updates.md`.
4. If Dev, Important, or GD needs to act, write a note in the matching inbox file.

## Chat Roles

- `Important`: project memory, summaries, status, and coordination.
- `Dev`: main "Очень Большая Бродилка" prototype in `/Users/qooobooo/Game Dev/Very Big Adventure/very-big-adventure`.
- `GD`: game design for "Очень Большая Бродилка"; owns rules, contracts, balance, core loop, and implementation-ready design notes for Dev.
